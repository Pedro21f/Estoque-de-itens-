# Sistema de Estoque — Python

Sistema simples de gerenciamento de estoque via terminal, desenvolvido como projeto prático de estudos em Python.

Este foi meu primeiro projeto aplicando conceitos de **dicionários**, **listas**, **tuplas**, **loops** e **entrada de dados** (`input`) em um programa funcional.

---

##  Sobre o projeto

O programa simula o controle de estoque de uma loja de roupas e calçados. Os produtos são armazenados em um dicionário, onde cada código identifica um item com nome, marca, quantidade e preço.

O usuário interage por um menu no terminal e pode adicionar, remover, atualizar e visualizar produtos.

---

##  Funcionalidades

-  **Adicionar produto** — cadastra um novo item no estoque
-  **Remover produto** — exclui um produto pelo código
-  **Atualizar produto** — altera as informações de um item existente
-  **Ver estoque** — exibe todos os produtos em formato de tabela
-  **Sair** — encerra o programa

---

## Conceitos de Python utilizados

| Conceito | Aplicação no projeto |
|----------|----------------------|
| Dicionários | Armazenamento dos produtos (`estoque`) |
| Listas | Montagem dos dados para a tabela |
| Loops `for` | Percorrer itens do estoque |
| Loop `while` | Menu principal contínuo |
| Condicionais `if/elif/else` | Validação de opções e produtos |
| `input()` | Interação com o usuário |
| Biblioteca externa | `tabulate` para exibir tabelas formatadas |
